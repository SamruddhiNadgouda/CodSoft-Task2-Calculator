# 🧮 Calculator — CodSoft Task 2

An interactive browser-based calculator built using **HTML**, **CSS**, and **JavaScript**, developed as part of the **CodSoft Web Development Internship** (Task 2).

---

## 📌 Project Overview

This project is a fully functional calculator web app that supports basic arithmetic operations. It features a clean button grid layout with a display screen, number keys, operator keys, and utility keys — all styled with CSS and powered by JavaScript logic.

---

## ✨ Features

- **Number Input** — Buttons for digits `0` through `9` and a decimal point `.`
- **Arithmetic Operations** — Addition `+`, Subtraction `-`, Multiplication `*`, Division `/`
- **Percentage** — `%` key for percentage calculations
- **All Clear** — `AC` button to reset the display completely
- **Delete** — `DE` button to remove the last entered character
- **Equals** — `=` button to evaluate and display the result
- **Live Display** — Text input field that shows the current expression and result

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Calculator structure and button layout |
| CSS3 | Styling, grid layout, and visual design |
| JavaScript | Button click logic, expression evaluation, AC/DE handling |
